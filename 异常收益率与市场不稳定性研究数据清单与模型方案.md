## 一、研究框架

### 1.1 核心逻辑

```
经济金融冲击(X) → 异常收益率(Y) → 市场不稳定性评估
```

**因变量(Y)**：异常收益率（Abnormal Return, AR）
- 反映市场对信息的"超预期反应"
- 绝对值|AR|越大，代表市场定价偏离正常水平越严重，不稳定性越高

**研究目标**：识别哪些宏观/金融因素会显著放大市场的异常反应，从而预警不稳定性

---

## 二、因变量(Y)定义与计算

### 2.1 异常收益率的多种计算方式

| 方法 | 公式 | 适用场景 | 数据需求 |
|------|------|---------|---------|
| **市场调整法** | AR_it = R_it - R_mt | 快速估算 | 个股/指数+市场收益率 |
| **市场模型** | AR_it = R_it - (α̂ + β̂·R_mt) | 标准方法 | 估计期历史数据 |
| **CAPM模型** | AR_it = R_it - [Rf + β·(R_m - Rf)] | 理论严谨 | 无风险利率 |
| **Fama-French三因子** | AR = R - [Rf + β·RMRF + s·SMB + h·HML] | 控制规模价值 | 三因子数据 |
| **GARCH条件收益** | AR_it = R_it - μ_t(σ²_t) | 波动聚集场景 | 需估计GARCH |

### 2.2 不稳定性度量方式

| 指标 | 计算 | 含义 |
|------|------|------|
| **绝对异常收益** | \|AR\| | 偏离正常水平程度 |
| **异常收益波动** | AR的滚动标准差 | 异常收益的不稳定性 |
| **极端异常收益** | AR的95%/99%分位数 | 尾部风险 |
| **异常收益离散度** | 个股AR的横截面标准差 | 市场分化程度 |

## 四、变量筛选与预处理

### 4.1 变量筛选原则

1. **理论基础**：变量需要有明确的经济学解释
2. **数据质量**：缺失率<10%，时间跨度足够
3. **多重共线性**：VIF<10，相关系数<0.8
4. **预测能力**：单变量回归p值<0.1

### 4.2 降维方法

```
主成分分析(PCA) → 提取宏观经济因子、金融条件因子、情绪因子
动态因子模型(DFM) → 提取共同因子
LASSO回归 → 自动变量选择
```

### 4.3 数据预处理

| 步骤 | 处理方法 | 目的 |
|------|---------|------|
| **缺失值** | 线性插值/EM算法 | 补全数据 |
| **异常值** | Winsorize(1%-99%) | 防止极端值主导 |
| **标准化** | Z-score标准化 | 消除量纲差异 |
| **滞后处理** | 宏观指标滞后1期 | 解决公布时滞 |
| **频率对齐** | 月→日：前向填充 | 混频数据处理 |

## 五、回归模型方案

### 5.1 基础模型

#### 模型1：多元线性回归（基准）

```
|AR_t| = α + Σβᵢ·Xᵢ,t + ε_t
```

**因变量**：异常收益率绝对值（\|AR\|）代表市场偏离正常定价的程度

**适用场景**：初步识别显著影响因素

---

#### 模型2：分位数回归（捕捉极端情况）

```
Q_{|AR|}(τ|X) = X'β(τ),  τ ∈ {0.5, 0.75, 0.9, 0.95}
```

**优势**：识别对极端异常收益（如τ=0.95）影响更大的因素

**解释**：哪些因素在市场高度不稳定时变得更加重要

---

#### 模型3：动态面板回归（时变影响）

```
|AR_t| = α + β₁·X_t + β₂·(X_t × Crisis_Dummy) + γ·|AR_{t-1}| + ε_t
```

**交互项**：检验金融危机/高波动期间，因素影响是否增强

**滞后因变量**：捕捉异常收益的持续性

---

### 5.2 时间序列模型

#### 模型4：ARDL自回归分布滞后

```
|AR_t| = α + Σφⱼ·|AR_{t-j}| + ΣΣθᵢₖ·X_{i,t-k} + ε_t
```

**优势**：捕捉宏观因素对异常收益的滞后影响

**边界检验**：长期均衡关系检验（Pesaran et al.）

---

#### 模型5：GARCH-M模型（波动反馈）

```
|AR_t| = μ + δ·σ_t + Σβᵢ·Xᵢ,t + ε_t
σ²_t = ω + α·ε²_{t-1} + β·σ²_{t-1} + Σγⱼ·Xⱼ,t
```

**GARCH-in-Mean**：检验波动率是否直接影响异常收益水平

**外生变量**：宏观因素进入波动率方程

---

### 5.3 非线性模型

#### 模型6：门槛回归（Threshold Regression）

```
|AR_t| = {
         α₁ + β₁·X_t + ε,  if q_t ≤ γ
         α₂ + β₂·X_t + ε,  if q_t > γ
       }
```

**门槛变量q**：市场状态（如VIX、ERP）

**目的**：识别"低波动状态"与"高波动状态"下影响因素的不同

---

#### 模型7：马尔可夫区制转移模型（MS-AR）

```
|AR_t| = μ_{s_t} + Σβᵢ,s_t·Xᵢ,t + ε_t
P(s_t = j | s_{t-1} = i) = pᵢⱼ
```

**区制s**：{低不稳定, 高不稳定}

**目的**：识别市场在不同"体制"间切换的规律，以及各状态下关键驱动因素

---

### 5.4 机器学习方法（可解释性）

#### 模型8：LASSO/Ridge回归

```
min:  Σ(|AR_t| - X'β)² + λ·Σ|βⱼ|
```

**用途**：高维变量自动选择，识别核心预测因子

---

#### 模型9：随机森林 + SHAP分析

```
|AR| = f(X) + ε,  f为非线性函数
```

**SHAP值**：量化每个变量对预测的贡献度

**优势**：捕捉复杂非线性关系，同时保持可解释性

---

### 5.5 网络/溢出模型

#### 模型10：Diebold-Yilmaz溢出指数

```
X_t = Φ·X_{t-1} + ε_t
Forecast Error Variance Decomposition → Spillover Index
```

**用途**：量化不同市场/行业间的风险传染强度

## 六、模型诊断与稳健性

### 6.1 诊断检验清单

| 检验类型 | 方法 | 目的 |
|---------|------|------|
| **平稳性** | ADF, KPSS, PP | 避免伪回归 |
| **协整** | Johansen, EG两步法 | 长期均衡关系 |
| **异方差** | ARCH-LM, White检验 | 是否存在波动聚集 |
| **序列相关** | Ljung-Box, DW检验 | 残差是否自相关 |
| **多重共线性** | VIF, 条件指数 | 变量间相关性 |
| **内生性** | Hausman检验 | 是否需要IV/GMM |
| **结构断点** | Chow检验, Bai-Perron | 是否存在体制转换 |

### 6.2 稳健性检验方案

1. **替换因变量**：使用AR²、|AR|的滚动标准差、极端分位数替代
2. **替换估计方法**：OLS → 稳健回归 → 分位数回归
3. **样本外检验**：滚动窗口预测，评估模型泛化能力
4. **工具变量**：对可能存在内生性的变量使用IV-2SLS
5. **替代变量**：同一概念使用不同代理变量重新估计

## 七、预期结果与解释

### 7.1 核心假设

| 假设 | 内容 | 预期符号 | 经济含义 |
|------|------|---------|---------|
| H1 | 高估值(低ERP) → \|AR\|↑ | β_ERP < 0 | 估值极端→均值回归压力 |
| H2 | 高杠杆 → \|AR\|↑ | β_Lev > 0 | 杠杆放大波动 |
| H3 | 低流动性 → \|AR\|↑ | β_Amihud > 0 | 流动性枯竭→价格冲击 |
| H4 | 高情绪 → \|AR\|↑ | β_Sentiment > 0 | 情绪极端→反转风险 |
| H5 | 政策不确定性↑ → \|AR\|↑ | β_EPU > 0 | 政策不确定→定价困难 |
| H6 | VIX↑ → \|AR\|↑ | β_VIX > 0 | 全球恐慌传导 |
| H7 | 信用利差↑ → \|AR\|↑ | β_CreditSpread > 0 | 信用恶化→风险溢价重估 |

### 7.2 输出指标

1. **显著因子识别**：各变量对\|AR\|的弹性系数及显著性
2. **解释力**：R²（模型解释异常收益波动的比例）
3. **状态依赖**：高/低波动状态下影响因素的差异
4. **预测能力**：样本外MAE/RMSE
5. **风险贡献度**：各因素对极端\|AR\|的贡献分解

### 7.3 实际应用

1. **市场风险预警系统**：基于实时X变量预测\|AR\|，超过阈值时预警
2. **压力测试**：模拟极端情景（如VIX翻倍、信用利差走阔）下的市场反应
3. **投资策略**：基于\|AR\|预测构建波动率交易策略
4. **政策评估**：量化特定政策对市场稳定性的影响

```python
# 假设已收集所有变量为DataFrame: df
# 列包括: date, AR, GDP, CPI, M2, VIX, ERP, Amihud, ...

# 计算异常收益绝对值
df['abs_AR'] = df['AR'].abs()

# 处理缺失值
df = df.fillna(method='ffill').dropna()

# 标准化（除虚拟变量外）
scaler = StandardScaler()
X_cols = [col for col in df.columns if col not in ['date', 'AR', 'abs_AR']]
df[X_cols] = scaler.fit_transform(df[X_cols])

# ========== 2. 多元线性回归 ==========
X = sm.add_constant(df[X_cols])
y = df['abs_AR']

model_ols = sm.OLS(y, X).fit(cov_type='HAC', cov_kwds={'maxlags': 5})
print(model_ols.summary())

# ========== 3. 分位数回归 ==========
from statsmodels.regression.quantile_regression import QuantReg

quantiles = [0.5, 0.75, 0.9, 0.95]
for q in quantiles:
    model_qr = QuantReg(y, X).fit(q=q)
    print(f"\n分位数 q={q}:")
    print(model_qr.summary())

# ========== 4. GARCH-M模型 ==========
# 构建外生变量矩阵
exog_vars = ['VIX', 'ERP', 'Amihud', 'CreditSpread']
exog = sm.add_constant(df[exog_vars])

# GARCH(1,1) with exogenous variables in variance
garch_model = arch_model(df['AR'], vol='Garch', p=1, q=1,
                          dist='t', rescale=False)
garch_result = garch_model.fit()
print(garch_result.summary())

# ========== 5. LASSO变量选择 ==========
lasso = LassoCV(cv=5, random_state=42)
lasso.fit(df[X_cols], y)

# 识别重要变量
important_vars = pd.Series(lasso.coef_, index=X_cols)
important_vars = important_vars[important_vars != 0].sort_values(key=abs, ascending=False)
print("LASSO选择的显著变量:")
print(important_vars)

# ========== 6. 门槛回归 ==========
from statsmodels.tsa.threshold_model import ThresholdRegression

# 以VIX为门槛变量
th_model = ThresholdRegression(y, df[X_cols], th_var=df['VIX'])
th_result = th_model.fit()
print(th_result.summary())

# ========== 7. 滚动预测评估 ==========
from sklearn.metrics import mean_absolute_error, r2_score

window = 252  # 1年滚动窗口
predictions = []
actuals = []

for i in range(window, len(df)):
    train = df.iloc[i-window:i]
    test = df.iloc[i:i+1]

    X_train = sm.add_constant(train[X_cols])
    y_train = train['abs_AR']

    model_roll = sm.OLS(y_train, X_train).fit()
    pred = model_roll.predict(sm.add_constant(test[X_cols]))

    predictions.append(pred.values[0])
    actuals.append(test['abs_AR'].values[0])

print(f"样本外MAE: {mean_absolute_error(actuals, predictions)}")
print(f"样本外R²: {r2_score(actuals, predictions)}")
```
