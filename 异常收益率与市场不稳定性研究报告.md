# 异常收益率驱动的金融市场不稳定性评估研究

---

## 摘要

本研究旨在构建基于异常收益率的金融市场不稳定性评估框架，通过识别驱动市场异常反应的关键宏观与金融因素，建立市场不稳定性的预警机制。研究以异常收益率（Abnormal Return, AR）作为核心因变量，采用市场模型、CAPM模型及Fama-French三因子模型等多种方法计算得到；自变量体系涵盖宏观经济基本面、金融市场结构、流动性与交易行为、波动性与风险、市场情绪与行为、政策与制度环境、国际溢出与外部环境等七大维度共计60余项指标。

研究方法上，本研究构建了包含基础线性模型（OLS、分位数回归）、时间序列模型（ARDL、GARCH-M）、非线性模型（门槛回归、马尔可夫区制转移）以及机器学习方法（LASSO、随机森林）的多层次模型体系。通过多种模型的交叉验证与稳健性检验，识别对市场不稳定性具有显著预测能力的关键因子。

## 一、研究描述

### 1.1 研究背景与问题提出

金融市场稳定性是维护金融体系健康运行的核心议题。2008年全球金融危机、2015年中国股市异常波动以及2020年新冠疫情引发的市场剧烈震荡等事件表明，市场在面临外部冲击时往往表现出非理性的过度反应，这种异常反应是引发市场不稳定的直接诱因。传统的风险度量指标如波动率、VaR等主要反映市场已实现的风险水平，而对于市场为何会在特定时点产生异常反应、哪些因素会放大这种异常反应等前瞻性问题的研究仍显不足。

异常收益率（Abnormal Return）作为事件研究法的核心指标，反映了资产收益率超出正常预期收益的部分，是市场对信息冲击"超预期反应"的量化体现。当异常收益率的绝对值显著增大时，意味着市场定价偏离正常水平，市场不稳定性随之上升。因此，研究异常收益率的驱动因素，对于理解市场不稳定的形成机制、构建前瞻性风险预警系统具有重要的理论价值与实践意义。

### 1.2 研究目标

本研究设定以下三个层次的研究目标：

**目标一：识别关键驱动因素**

从宏观经济基本面、金融市场结构、流动性与交易行为、市场情绪与行为、政策与制度环境、国际溢出效应等多维度，识别对异常收益率具有显著解释力的核心因子，明确各因子对市场不稳定性的影响方向与程度。

**目标二：刻画非线性关系与状态依赖**

检验影响因素与异常收益率之间是否存在非线性关系，探究在市场高波动状态与低波动状态下，各因素影响作用是否存在显著差异，揭示市场不稳定的异质性特征。

**目标三：构建预测与预警框架**

基于识别出的关键因子与建立的模型关系，构建市场不稳定性的实时评估与预警系统，为投资者风险管理与监管者宏观审慎决策提供量化依据。

### 1.3 核心研究逻辑

本研究的核心逻辑框架如下：

```
经济金融冲击(X) → 异常收益率(Y) → 市场不稳定性评估
```

**因变量(Y)**：异常收益率（Abnormal Return, AR）及其绝对值|AR|。异常收益率的绝对值越大，代表市场定价偏离正常水平越严重，市场不稳定性越高。

**自变量(X)**：涵盖七大维度的宏观与金融指标，代表可能引发市场异常反应的各类冲击。

**研究目标**：识别哪些宏观/金融因素会显著放大市场的异常反应，从而预警不稳定性。

### 1.4 研究创新点

本研究的创新之处体现在以下三个方面：

1. **指标构建创新**：系统性地将异常收益率作为市场不稳定性的核心代理变量，突破了传统研究仅关注波动率等二阶矩指标的局限，从收益率一阶矩偏离的角度刻画市场不稳定。

2. **维度整合创新**：构建了涵盖宏观经济、金融市场、微观结构、投资者行为、政策环境、国际溢出等多维度的综合指标体系，实现了对市场不稳定性的全景式分析。

3. **方法体系创新**：整合传统计量经济学方法（分位数回归、GARCH族模型、门槛回归、区制转移模型）与现代机器学习方法（LASSO、随机森林、SHAP分析），形成互补的模型体系。

## 二、文献综述

### 2.1 异常收益率与市场有效性研究

异常收益率的概念源于Fama等人（1969）开创性的事件研究法（Event Study Methodology），该方法通过比较实际收益率与正常预期收益率的差异来识别市场对特定事件的反应。Fama（1970）的有效市场假说（EMH）指出，在有效市场中，资产价格应充分反映所有可得信息，异常收益率应随机出现且均值为零。然而，大量实证研究表明，市场并非完全有效，存在着各种异象（Anomalies）。

### 2.2 市场不稳定性的度量与影响因素

市场不稳定性的度量方法经历了从简单到复杂、从静态到动态的演进过程。早期的研究主要采用方差或标准差等简单统计量度量波动（Markowitz, 1952）。Engle（1982）提出的ARCH模型及其拓展形式GARCH模型（Bollerslev, 1986）为波动率的时变特征建模提供了有力工具。随后，RiskMetrics方法、已实现波动率（Realized Volatility, Andersen et al., 2001）等高频数据方法相继涌现。

关于市场不稳定性的影响因素，学术文献主要从以下几个维度展开：

**宏观经济因素**：GDP增长率、通货膨胀率、利率水平、货币供应量等宏观变量对市场稳定性的影响已得到广泛验证（Schwert, 1989; Hamilton & Lin, 1996）。研究表明，经济衰退期市场波动性显著上升，货币政策的不确定性也会增加市场不稳定。

**市场微观结构因素**：流动性（Amihud, 2002）、市场深度、买卖价差等微观结构指标与市场稳定性密切相关。流动性枯竭往往是市场危机的重要特征（Brunnermeier & Pedersen, 2009）。

**投资者行为因素**：行为金融学研究表明，投资者情绪（Baker & Wurgler, 2006）、过度交易、羊群效应等行为因素会导致市场价格偏离基本面，增加市场不稳定。

**国际溢出因素**：在全球化背景下，跨国市场间的风险传染日益显著（Diebold & Yilmaz, 2012）。美国市场的波动通过贸易渠道、金融渠道和情绪渠道传导至其他市场。

### 2.3 计量经济学方法在金融研究中的应用

**分位数回归**：Koenker & Bassett（1978）提出的分位数回归方法为研究不同市场状态下的异质性关系提供了工具。Baur（2012）等学者应用分位数回归研究了极端市场条件下的风险因子表现。

**GARCH族模型**：GARCH-in-Mean模型（GARCH-M）由Engle等人（1987）提出，允许条件波动率进入均值方程，用于检验风险溢价假说。

**门槛回归与区制转移模型**：Hansen（1999）的门槛回归方法可以捕捉变量关系的结构性变化。Hamilton（1989）的马尔可夫区制转移模型（MS-AR）被广泛应用于识别金融市场在不同体制间的转换规律。

**机器学习方法**：近年来，LASSO（Tibshirani, 1996）、随机森林（Breiman, 2001）等机器学习方法在金融预测领域得到越来越多的应用。SHAP值（Lundberg & Lee, 2017）为黑箱模型提供了可解释性分析工具。

### 2.4 文献述评与研究空间

综观现有文献，虽然关于市场不稳定性的研究已经取得了丰富的成果，但仍存在以下研究空间：

1. 现有研究多以波动率等二阶矩指标作为不稳定性的代理变量，而对异常收益率一阶矩偏离的关注相对不足；

2. 大多数研究侧重于单一维度或少数几个因素的影响分析，缺乏多维度、系统性的综合框架；

3. 传统计量方法与机器学习方法在金融研究中往往被割裂使用，缺乏系统性的整合与比较。

本研究正是在上述背景下，试图构建基于异常收益率的市场不稳定性评估框架，整合多维度影响因素与多种计量方法，为理解市场不稳定的形成机制提供新的视角。

#### 3.2.2 异常收益率计算方法

**方法一：市场调整法（Market-Adjusted Model）**

$$AR_{it} = R_{it} - R_{mt}$$

其中，$R_{it}$为个股$i$在$t$日的收益率，$R_{mt}$为市场组合（沪深300）在$t$日的收益率。

**方法二：市场模型（Market Model）**

$$AR_{it} = R_{it} - (\hat{\alpha}_i + \hat{\beta}_i \cdot R_{mt})$$

其中，$\hat{\alpha}_i$和$\hat{\beta}_i$通过估计期（如过去250个交易日）的OLS回归得到。

**方法三：CAPM模型**

$$AR_{it} = R_{it} - [R_{ft} + \hat{\beta}_i \cdot (R_{mt} - R_{ft})]$$

**方法四：Fama-French三因子模型**

$$AR_{it} = R_{it} - [R_{ft} + \beta_i^{MKT} \cdot RMRF_t + \beta_i^{SMB} \cdot SMB_t + \beta_i^{HML} \cdot HML_t]$$

其中，$RMRF$为市场风险溢价，$SMB$为市值因子，$HML$为价值因子。

### 3.4 数据预处理方案

#### 3.4.1 缺失值处理

| 缺失情况 | 处理方法 | 适用变量 |
|---------|---------|---------|
| 随机缺失 | 线性插值法 | 日度高频数据 |
| 连续缺失<5% | 前后向填充 | 宏观指标 |
| 缺失严重(>20%) | 删除该变量 | - |
| 公布时滞 | 滞后1期处理 | 月度宏观数据 |

#### 3.4.2 异常值处理

采用Winsorize缩尾处理，对所有连续变量在1%和99%分位数处进行截断，防止极端值主导回归结果。

$$X_{winsorized} = \begin{cases} X_{1\%}, & \text{if } X < X_{1\%} \\ X_{99\%}, & \text{if } X > X_{99\%} \\ X, & \text{otherwise} \end{cases}$$

#### 3.4.3 频率对齐与标准化

| 步骤 | 处理方法 | 目的 |
|------|---------|------|
| 频率对齐 | 月度→日度：前向填充 | 混频数据处理 |
| 滞后处理 | 宏观指标滞后1期 | 解决公布时滞 |
| 标准化 | Z-score标准化 | 消除量纲差异 |

标准化公式：
$$Z_i = \frac{X_i - \bar{X}}{\sigma_X}$$

#### 3.4.4 多重共线性检验

计算方差膨胀因子（VIF）进行变量筛选：
$$VIF_j = \frac{1}{1-R_j^2}$$

剔除VIF>10或两两相关系数>0.8的变量，确保模型估计的有效性。

## 五、回归建模

### 5.1 模型体系概览

本研究构建多层次模型体系，从基础线性关系到复杂非线性关系，从单一因素到多因素交互，从传统计量到机器学习方法，全面探究异常收益率的驱动机制。

| 模型类别 | 具体模型 | 主要用途 |
|---------|---------|---------|
| 基础线性模型 | OLS回归、分位数回归 | 识别显著因素，捕捉极端影响 |
| 时间序列模型 | ARDL、GARCH-M | 刻画动态滞后关系，波动反馈 |
| 非线性模型 | 门槛回归、MS-AR | 识别状态依赖，体制转换 |
| 机器学习 | LASSO、随机森林 | 高维变量选择，非线性预测 |

### 5.2 基础线性模型

#### 5.2.1 多元线性回归（基准模型）

**模型设定**：

$$|AR_t| = \alpha + \sum_{i=1}^{k} \beta_i X_{i,t} + \varepsilon_t$$

其中，$|AR_t|$为$t$期异常收益率的绝对值，$X_{i,t}$为第$i$个自变量，$\varepsilon_t$为误差项。

**估计方法**：
- 采用OLS估计
- 使用异方差自相关一致（HAC）标准误（Newey-West标准误）
- 最大滞后阶数设为5阶

**预期结果解读**：
- 系数$\beta_i$表示自变量$X_i$每增加1个标准差，异常收益率绝对值变化$\beta_i$个单位
- $R^2$表示模型解释异常收益率波动的比例
- 基于t统计量判断显著性

**【估计结果待补充】**：实际回归系数、标准误、R²等结果待数据获取后补充。

#### 5.2.2 分位数回归（Quantile Regression）

**模型设定**：

$$Q_{|AR|}(\tau|X) = X'\beta(\tau), \quad \tau \in \{0.5, 0.75, 0.9, 0.95\}$$

其中，$Q_{|AR|}(\tau|X)$表示在给定$X$条件下$|AR|$的$\tau$分位数。

**估计方法**：
- 采用线性规划方法求解
- 估计中位数（$\tau=0.5$）、上四分位数（$\tau=0.75$）、90%分位数（$\tau=0.9$）、95%分位数（$\tau=0.95$）

**预期结果解读**：
- 比较不同分位数下各因素的系数差异
- 识别对极端异常收益影响更大的因素
- 例如：若VIX在$\tau=0.95$处的系数显著大于$\tau=0.5$，说明VIX在市场高度不稳定时作用更强

**【估计结果待补充】**：实际分位数回归结果待数据获取后补充。

### 5.3 时间序列模型

#### 5.3.1 ARDL自回归分布滞后模型

**模型设定**：

$$|AR_t| = \alpha + \sum_{j=1}^{p} \phi_j |AR_{t-j}| + \sum_{i=1}^{k} \sum_{l=0}^{q_i} \theta_{i,l} X_{i,t-l} + \varepsilon_t$$

其中，$p$为自回归阶数，$q_i$为第$i$个自变量的滞后阶数。

**预期分析内容**：
- 异常收益率的自相关性检验
- 宏观因素对异常收益的滞后影响（货币政策的时滞效应）
- 长期均衡关系检验（Pesaran边界检验）

**【估计结果待补充】**：实际ARDL模型估计结果待数据获取后补充。

#### 5.3.2 GARCH-M模型

**模型设定**：

均值方程：
$$AR_t = \mu + \delta \sigma_t + \sum_{i=1}^{k} \beta_i X_{i,t} + \varepsilon_t$$

方差方程：
$$\sigma_t^2 = \omega + \alpha \varepsilon_{t-1}^2 + \beta \sigma_{t-1}^2 + \sum_{j=1}^{m} \gamma_j Z_{j,t}$$

其中，$\sigma_t$为条件波动率，$\delta$检验波动率是否直接影响异常收益水平（GARCH-in-Mean效应）。

**预期分析内容**：
- 波动率持续性（$\alpha+\beta$接近1表示高持续性）
- 波动率反馈效应（$\delta$的符号与显著性）
- 外生变量对波动率的影响（$\gamma_j$）

**【估计结果待补充】**：实际GARCH-M模型估计结果待数据获取后补充。

### 5.4 非线性模型

#### 5.4.1 门槛回归（Threshold Regression）

**模型设定**：

$$|AR_t| = \begin{cases} \alpha_1 + \beta_1' X_t + \varepsilon_t, & \text{if } q_t \leq \gamma \\ \alpha_2 + \beta_2' X_t + \varepsilon_t, & \text{if } q_t > \gamma \end{cases}$$

其中，$q_t$为门槛变量（如VIX、ERP等市场状态指标），$\gamma$为门槛值。

**预期分析内容**：
- 检验门槛效应是否显著（Hansen检验）
- 估计门槛值$\gamma$
- 比较两区间的系数差异（$\beta_1$ vs $\beta_2$）
- 识别"低波动状态"与"高波动状态"下影响因素的不同

**【估计结果待补充】**：实际门槛回归结果待数据获取后补充。

#### 5.4.2 马尔可夫区制转移模型（MS-AR）

**模型设定**：

$$|AR_t| = \mu_{s_t} + \sum_{i=1}^{k} \beta_{i,s_t} X_{i,t} + \varepsilon_t$$

$$P(s_t = j | s_{t-1} = i) = p_{ij}$$

其中，$s_t \in \{1, 2\}$为区制变量（1=低不稳定状态，2=高不稳定状态），$p_{ij}$为转移概率。

**预期分析内容**：
- 估计各区制的参数（$\mu_1$ vs $\mu_2$，$\beta_{i,1}$ vs $\beta_{i,2}$）
- 估计转移概率矩阵
- 计算区制持续期期望（$E[duration|s=i] = 1/(1-p_{ii})$）
- 绘制平滑概率图，识别历史上的高不稳定期

**【估计结果待补充】**：实际MS-AR模型估计结果待数据获取后补充。

## 六、机器学习建模

### 6.1 LASSO回归（变量选择）

**模型设定**：

$$\min_{\beta} \sum_{t=1}^{T} (|AR_t| - X_t'\beta)^2 + \lambda \sum_{j=1}^{k} |\beta_j|$$

其中，$\lambda$为正则化参数，通过交叉验证选择。

**预期分析内容**：
- 通过交叉验证选择最优$\lambda$
- 识别被压缩至零的变量（不重要）
- 保留的非零系数变量即为核心预测因子
- 比较LASSO选择与OLS显著性的一致性

**【估计结果待补充】**：实际LASSO回归结果待数据获取后补充。

### 6.2 随机森林（Random Forest）

**模型设定**：

$$|AR| = f(X) + \varepsilon$$

其中，$f(\cdot)$为非线性函数，通过集成多棵决策树学习得到。

**模型参数**：
- 树的数量：n_estimators=500
- 最大深度：max_depth=None（自动生长）
- 特征采样比例：max_features='sqrt'
- 最小分裂样本数：min_samples_split=5

**SHAP值解释**：

使用SHAP（SHapley Additive exPlanations）值量化每个变量对预测的贡献度：

$$\phi_j = \sum_{S \subseteq N \setminus \{j\}} \frac{|S|!(|N|-|S|-1)!}{|N|!} [f_{S \cup \{j\}}(x) - f_S(x)]$$

**预期分析内容**：
- 特征重要性排序（基于Gini不纯度减少）
- SHAP值分析各变量的边际贡献
- 部分依赖图（Partial Dependence Plot）刻画非线性关系
- 识别变量间的交互效应

**【估计结果待补充】**：实际随机森林与SHAP分析结果待数据获取后补充。

```python
import statsmodels.api as sm
from sklearn.linear_model import LinearRegression

def calculate_abnormal_returns(df_stock, df_market, method='market_model'):
    """
    计算异常收益率

    Parameters:
    -----------
    df_stock : DataFrame
        个股收益率数据，包含'return'列
    df_market : DataFrame
        市场收益率数据，包含'return'列
    method : str
        计算方法：'market_adjusted', 'market_model', 'capm', 'ff3'

    Returns:
    --------
    ar : Series
        异常收益率序列
    """

    if method == 'market_adjusted':
        # 市场调整法：AR = R_stock - R_market
        ar = df_stock['return'] - df_market['return']

    elif method == 'market_model':
        # 市场模型：AR = R_stock - (alpha + beta * R_market)
        # 估计期（使用前252个交易日）
        X = sm.add_constant(df_market['return'].iloc[-252:])
        y = df_stock['return'].iloc[-252:]

        model = sm.OLS(y, X).fit()
        alpha, beta = model.params['const'], model.params['return']

        # 计算异常收益率
        ar = df_stock['return'] - (alpha + beta * df_market['return'])

    elif method == 'capm':
        # CAPM模型：AR = R_stock - [Rf + beta * (R_market - Rf)]
        # 需要无风险利率数据
        rf = df_stock['rf'] if 'rf' in df_stock.columns else 0.02/252

        X = sm.add_constant(df_market['return'].iloc[-252:] - rf)
        y = df_stock['return'].iloc[-252:]

        model = sm.OLS(y, X).fit()
        beta = model.params[0]

        ar = df_stock['return'] - (rf + beta * (df_market['return'] - rf))

    elif method == 'ff3':
        # Fama-French三因子模型
        # 需要SMB、HML因子数据
        ff_data = get_ff3_factors()  # 自定义函数获取因子数据

        X = sm.add_constant(ff_data[['MKT', 'SMB', 'HML']])
        y = df_stock['return'] - df_stock['rf']

        model = sm.OLS(y, X).fit()
        expected_return = (model.params['const'] +
                          model.params['MKT'] * ff_data['MKT'] +
                          model.params['SMB'] * ff_data['SMB'] +
                          model.params['HML'] * ff_data['HML'])

        ar = df_stock['return'] - expected_return

    return ar

def calculate_instability_measures(ar_series, window=20):
    """
    计算市场不稳定性度量指标

    Parameters:
    -----------
    ar_series : Series
        异常收益率序列
    window : int
        滚动窗口期数

    Returns:
    --------
    instability : DataFrame
        各不稳定性指标
    """
    instability = pd.DataFrame(index=ar_series.index)

    # 绝对异常收益
    instability['abs_ar'] = ar_series.abs()

    # 异常收益波动（滚动标准差）
    instability['ar_vol'] = ar_series.rolling(window).std()

    # 极端异常收益（95%分位数，滚动计算）
    instability['ar_extreme'] = ar_series.abs().rolling(window*5).quantile(0.95)

    return instability
```

```python
    分位数回归
    """
    X = sm.add_constant(X)
    results = {}

        results[f'q{int(q*100)}'] = model

def run_garch_m(ar_series, exog_vars=None):
    """
    GARCH-M模型估计
    【注】：需要安装arch包: pip install arch
    """
    # GARCH(1,1) with Student-t distribution
    am = arch_model(ar_series, vol='Garch', p=1, q=1,
                    dist='t', rescale=False)

    if exog_vars is not None:
        # 在波动率方程中加入外生变量
        am.volatility = arch_model(vol='Garch', p=1, q=1,
                                   external_regressors=exog_vars)

    return result

def run_lasso_selection(X, y):
    """
    LASSO变量选择
    """
    from sklearn.linear_model import LassoCV
    from sklearn.preprocessing import StandardScaler

    # 标准化
    scaler = StandardScaler()
    X_scaled = scaler.fit_transform(X)

    # LASSO交叉验证
```

```python
def diagnostic_tests(model, y, X):
    """
    模型诊断检验
    """
    from statsmodels.stats.diagnostic import (
        het_breuschpagan, acorr_ljungbox, linear_reset
    )
    from statsmodels.tsa.stattools import adfuller

    results = {}

    # 1. 异方差检验（White/BP检验）
    bp_test = het_breuschpagan(model.resid, X)
    results['heteroskedasticity'] = {
        'LM': bp_test[0],
        'p-value': bp_test[1]
    }

    # 2. 序列相关检验（Ljung-Box）
    lb_test = acorr_ljungbox(model.resid, lags=5)
    results['serial_correlation'] = lb_test

    # 3. 平稳性检验（ADF）
    adf_test = adfuller(y)
    results['stationarity'] = {
        'ADF': adf_test[0],
        'p-value': adf_test[1]
    }

    return results

def calculate_vif(X):
    """
    计算方差膨胀因子（VIF）
    """
    from statsmodels.stats.outliers_influence import variance_inflation_factor

    X_with_const = sm.add_constant(X)
    vif = pd.Series(
        [variance_inflation_factor(X_with_const.values, i)
         for i in range(X_with_const.shape[1])],
        index=X_with_const.columns
    )

    return vif
```

```python
from sklearn.metrics import mean_absolute_error, mean_squared_error, r2_score

def rolling_forecast(y, X, window=252):
    """
    滚动窗口预测

    Parameters:
    -----------
    y : Series
        因变量
    X : DataFrame
        自变量
    window : int
        滚动窗口大小

    Returns:
    --------
    predictions : Series
        预测值
    actuals : Series
        实际值
    """
    predictions = []
    actuals = []
    dates = []

    for i in range(window, len(y)):
        # 训练集
        y_train = y.iloc[i-window:i]
        X_train = X.iloc[i-window:i]

        # 测试集
        y_test = y.iloc[i]
        X_test = X.iloc[i:i+1]

        # 估计模型
        model = sm.OLS(y_train, sm.add_constant(X_train)).fit()

        # 预测
        pred = model.predict(sm.add_constant(X_test))

        predictions.append(pred.values[0])
        actuals.append(y_test)
        dates.append(y.index[i])

    predictions = pd.Series(predictions, index=dates)
    actuals = pd.Series(actuals, index=dates)

    # 计算评估指标
    mae = mean_absolute_error(actuals, predictions)
    rmse = np.sqrt(mean_squared_error(actuals, predictions))
    r2 = r2_score(actuals, predictions)

    print(f"样本外MAE: {mae:.4f}")
    print(f"样本外RMSE: {rmse:.4f}")
    print(f"样本外R²: {r2:.4f}")

    return predictions, actuals
```

```r
calculate_spillover <- function(data, n_ahead = 10) {
  library(vars)

  # VAR估计
  var_model <- VAR(data, p = 4, type = "const")

  # 预测误差方差分解
  fevd <- fevd(var_model, n.ahead = n_ahead)
```
