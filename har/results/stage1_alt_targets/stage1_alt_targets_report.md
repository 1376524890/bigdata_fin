# 第一阶段替代因变量实验报告

## 一、实验概述

本实验测试将第一阶段因变量从'未来收益率'改为'未来波动率/不稳定性/涨跌状态指标'后，
样本外表现是否更稳定。

**实验设置：**
- 样本期：2015-07-02 至 2025-12-25
- 训练测试切分：前60%训练，后40%测试
- 训练内部验证：前80%子训练，后20%验证
- 固定随机种子：42

## 二、哪一类因变量最容易预测？

基于 HARplusMacro_Ridge 模型的测试集平均 R² 排序：

1. 未来平均绝对收益：平均 R² = 0.0098
2. 未来实现波动率（取log）：平均 R² = -0.0195
3. 未来涨跌方向平衡度：平均 R² = -0.1446
4. 未来上涨天数占比：平均 R² = -0.1446

## 七、数值不稳定模型记录

所有模型数值稳定。

## 八、预测值超界检查

对于 upratio（理论范围 [0,1]）和 signbalance（理论范围 [-1,1]），
线性回归模型可能产生超界预测值。
这是线性模型的固有局限，不影响模型本身的有效性，
但实际应用时可能需要截断处理。

## 九、最终判断

### 结论：C

**即使改成状态目标，第一阶段改善仍有限，因此一阶段本身不应承担太强预测任务。**

- 替代目标虽优于收益率，但绝对解释力仍有限
- 第一阶段应定位为'基准建模'而非'精准预测'
- 重点放在第二阶段的偏离成因识别
