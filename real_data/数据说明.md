## 六、情绪指标说明

### 计算公式

```
sentiment_raw = momentum_5d × log(volume_ratio_5d)
sentiment_zscore = (sentiment_raw - 252日均值) / 252日标准差
```

### 等级划分

| z值范围 | 等级 | 含义 |
|---------|------|------|
| z > 2 | 极度贪婪 | 市场过热 |
| 1 < z ≤ 2 | 贪婪 | 偏乐观 |
| -1 ≤ z ≤ 1 | 中性 | 正常 |
| -2 ≤ z < -1 | 恐惧 | 偏悲观 |
| z < -2 | 极度恐惧 | 市场恐慌 |

## 七、使用建议

2. **时滞问题**: 宏观数据存在发布时滞，使用时需注意
3. **标准化**: 回归分析前建议对变量进行标准化处理
4. **起始时间**: 
   - 北向资金从 2014-11-17 开始（沪港通开通）
   - iVIX从 2015-02-09 开始（期权上市）
   - 融资融券从 2012-09-27 开始
