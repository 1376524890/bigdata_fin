# 宏观公告"预期差"效应的事件研究分析报告

## 一、研究概述

### 1.1 核心命题

传统研究关注宏观经济指标**水平值**对股市的影响，本研究聚焦于**
"预期修正"**（Expectation Revision）的市场定价机制：

> 股市对宏观公告的反应程度取决于**实际值与市场一致预期的偏离程度**，而非绝对水平。

### 1.2 理论框架

```
宏观公告效应 = f(预期差, 不确定性, 市场状态)
              = f(Surprise, Dispersion, Sentiment)
```

**有效市场假说**的推论：若市场有效，只有**未被预期到**的信息才会引起价格调整；已充分预期的信息应被提前消化，公告日不应产生显著超额收益。

### 1.3 研究意义

1. **学术价值**：检验中国股市的信息处理效率，验证是否存在"预期漂移"（Drift）或"过度反应"（Overreaction）
2. **实践价值**：为宏观事件驱动型交易策略提供量化依据

### 2.1 宏观数据分类与来源

| 数据类型 | 具体指标 | 发布频率 | 数据来源 |
|---------|---------|---------|---------|
| **经济增长** | GDP同比、工业增加值同比 | 季度/月度 | 国家统计局 + Wind一致预期 |
| **通货膨胀** | CPI同比、PPI同比、核心CPI | 月度 | Wind宏观数据库 |
| **货币政策** | MLF利率、LPR、存款准备金率 | 不定期 | 央行官网 + Wind预期 |
| **货币信贷** | M2增速、社融增量、新增贷款 | 月度 | 央行 + Wind预期 |
| **景气指数** | PMI（官方+财新） | 月度 | 国家统计局/财新 + Wind预期 |
| **就业数据** | 城镇调查失业率 | 月度 | 统计局 + 预期数据 |
| **贸易数据** | 进出口同比、贸易差额 | 月度 | 海关总署 + Wind预期 |

### 2.2 市场预期数据来源

#### 选项A：Wind一致预期（推荐）

- **数据说明**：Wind采集多家券商/机构对宏观指标的预测中位数/均值
- **字段**：预期均值、预期中位数、预期标准差、机构数量
- **获取方式**：Wind终端 - 宏观经济 - 经济数据预测
- **覆盖度**：主要指标（GDP、CPI、PPI、PMI、M2等）覆盖完整

#### 选项B：彭博一致预期

- **数据说明**：国际投资者视角的预期数据
- **优势**：可与A股外资持仓数据结合分析
- **局限**：需要彭博终端权限

#### 选项C：自主构建预期（降级方案）

若无法获取一致预期数据，可使用**ARMA模型**基于历史数据生成理性预期：

```
E[X_t] = ARMA(p,q)模型预测值
Surprise_t = X_t^{actual} - X_t^{forecast}
```

## 三、建模方法

### 3.1 研究设计：事件研究法

#### 时间窗口定义

```
        [-T1, -1]       [0]        [+1, +T2]
        ─────────      ───        ─────────
        估计窗口      事件日       事件窗口
        (Estimation)   (t=0)      (Event)
```

| 窗口类型 | 时间范围 | 用途 |
|---------|---------|------|
| **估计窗口** | [-60, -11] 或 [-120, -11] | 估计正常收益率模型参数 |
| **事件日** | t=0 | 宏观数据公布日 |
| **事件窗口** | [-3, +3] 或 [-1, +1] | 计算公告效应（CAR） |
| **事后窗口** | [+4, +20] | 检验漂移/反转效应 |

#### 正常收益率模型

**模型1：市场调整模型（最简单）**

```
AR_it = R_it - R_mt
```

**模型2：市场模型（推荐）**

```
R_it = α_i + β_i·R_mt + ε_it
AR_it = R_it - (α̂_i + β̂_i·R_mt)
```

**模型3：Fama-French三因子模型（稳健性）**

```
R_it = α_i + β_i·R_mt + s_i·SMB_t + h_i·HML_t + ε_it
```

#### 累计异常收益率(CAR)计算

```
CAR_i(t1, t2) = Σ AR_iτ,  τ ∈ [t1, t2]
```

### 3.2 核心回归模型

#### 模型1：预期差效应检验（主模型）

检验预期差大小是否显著影响市场反应强度：

```
CAR_i[0,+1] = α + β·Surprise_i + γ·|Surprise_i| + δ·Controls_i + ε_i
```

**变量说明**：
- `Surprise`：标准化预期差（正值=超预期，负值=低于预期）
- `|Surprise|`：预期差绝对值（捕捉反应强度非对称性）
- `Controls`：市场波动率(VIX)、事前不确定性(预期分歧度)、宏观经济周期虚拟变量

**预期结果**：
- β > 0：正向预期差带来正超额收益
- γ 检验非对称性：好消息 vs 坏消息的市场反应是否对称

#### 模型2：异质性反应检验（分样本）

```
CAR_i = α + β₁·Surprise·CPI_Dummy + β₂·Surprise·GDP_Dummy + β₃·Surprise·PMI_Dummy + Controls + ε
```

检验不同宏观指标的预期差效应是否存在差异。

#### 模型3：漂移效应检验（Post-Earnings Announcement Drift拓展）

```
CAR_i[+2,+20] = α + β·Surprise_i·CAR_i[0,+1] + Controls + ε
```

检验宏观公告后是否存在价格漂移（反应不足的证据）。

#### 模型4：门槛回归（非线性效应）

```
CAR = {
        α₁ + β₁·Surprise + ε,  if |Surprise| ≤ γ (阈值)
        α₂ + β₂·Surprise + ε,  if |Surprise| > γ
      }
```

检验是否存在"注意力门槛"——只有足够大的预期差才会引发市场反应。

### 3.3 稳健性检验

1. **安慰剂检验**：随机生成"伪公告日"，检验是否出现虚假显著性
2. **不同事件窗口**：[-1,+1]、[-3,+3]、[-5,+5]对比
3. **排除 confounding events**：剔除同日有其他重大政策发布的事件
4. **工具变量法**：使用外生冲击（如疫情封控）作为预期差的工具变量

## 五、期望结果

### 5.1 主要假设与预期发现

| 假设 | 内容 | 预期结果 | 理论含义 |
|-----|------|---------|---------|
| **H1: 预期差效应** | 正向预期差→正超额收益；负向预期差→负超额收益 | β_surprise > 0 (显著) | 市场对意外信息有效反应 |
| **H2: 非对称性** | 坏消息反应 > 好消息反应（损失厌恶） | \|β_negative\| > \|β_positive\| | 行为金融学：投资者风险厌恶 |
| **H3: 漂移效应** | 公告后存在价格漂移（反应不足） | CAR[+2,+20] 与 Surprise 正相关 | 有限注意力/信息逐步扩散 |
| **H4: 门槛效应** | 小的预期差被市场忽略，大的预期差才引发反应 | 存在显著门槛值 γ > 0 | 注意力约束下的非线性定价 |

### 5.2 定量输出指标

1. **事件日平均CAR**：
   - 正向意外组 CAR[0,+1] = +0.5%~+1.2%
   - 负向意外组 CAR[0,+1] = -0.8%~-1.5%

2. **预期差弹性系数**：
   - 预期差每增加1个标准差，CAR增加0.3%~0.8%

3. **漂移幅度**：
   - 事件窗口反应后，20天内漂移幅度达到事件窗口的30%~50%

4. **市场效率指标**：
   - 若CAR[+2,+20]与Surprise不相关 → 半强式有效
   - 若存在显著正相关 → 存在反应不足（套利机会）

### 5.3 可视化输出

1. **CAR累积路径图**：对比不同预期差组别的CAR动态
2. **散点图+拟合线**：Surprise vs CAR，观察线性/非线性关系
3. **热力图**：不同宏观指标 × 市场状态 的预期差效应矩阵
4. **门槛效应图**：展示不同Surprise区间的斜率变化

### 5.4 稳健性检验结果

- **安慰剂检验**：伪公告日的CAR均值应不显著异于0（p > 0.1）
- **不同估计期**：改变估计窗口后主要结论保持不变
- **剔除极端值**：Winsorize处理后结论稳健

### 5.5 实际应用价值

1. **事件驱动策略**：构建基于宏观预期差的短期交易策略
   - 信号：Surprise > 1.5σ 时做多，Surprise < -1.5σ 时做空
   - 持仓：公告日前1日至后3日

2. **宏观因子暴露**：为风险模型提供宏观事件风险因子

3. **政策评估**：评估市场对央行沟通/政策引导的理解程度

## 六、潜在问题与解决方案

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| **同日多事件** | 多指标同日公布（如GDP+工业增加值） | 剔除多事件同日样本，或分离出单一指标效应 |
| **提前泄露** | 数据在正式公布前已部分泄露 | 将事件窗口前移至[-5,-1]检验泄露效应 |
| **宏观相关性** | GDP与PMI高度相关，难以分离效应 | 使用偏相关系数或主成分分析降维 |
| **小样本问题** | 季度GDP数据点较少（约40个） | 结合月度指标（CPI、PMI）增加样本量 |
| **内生性** | 市场预期可能受前期股市影响 | 使用外生冲击作为工具变量 |

## 八、附录：关键概念解释

### 8.1 一致预期(Consensus Forecast)

多家机构对同一宏观指标的预测值的**中位数或均值**，代表市场主流预期。

### 8.2 预期差(Surprise)

```
Surprise = (Actual - Consensus) / Standard_Deviation(Forecasts)
```

标准化处理使得不同指标的预期差可比。

### 8.3 累计异常收益率(CAR)

```
CAR(t1, t2) = Σ_{τ=t1}^{t2} AR_τ
```

其中AR为异常收益率（实际收益-预期收益）。

### 8.4 漂移效应(Drift)

公告后价格继续朝同一方向移动的现象，表明市场对信息存在**反应不足**。

### 8.5 非对称反应(Asymmetric Response)

同等幅度的正向vs负向预期差，引发的市场反应幅度不同，通常负向意外的反应更强（损失厌恶）。
