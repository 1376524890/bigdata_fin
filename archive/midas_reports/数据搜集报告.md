## 一、数据搜集概述

### 1.1 研究数据需求

根据《异常收益率与市场不稳定性研究数据清单与模型方案》，需搜集以下类别数据：

| 类别 | 具体指标 | 优先级 |
|------|---------|--------|
| 因变量(Y) | 异常收益率(AR)、绝对异常收益率 | 核心 |
| 市场数据 | 指数价格、成交量、波动率 | 核心 |
| 宏观指标 | GDP、CPI、M2、利率等 | 重要 |
| 情绪指标 | 投资者情绪、恐慌贪婪指数 | 重要 |
| 资金流向 | 北向资金、融资融券 | 重要 |
| 国际数据 | VIX、标普500、美债收益率 | 补充 |

### 1.2 数据获取方式

| 方式 | 数据源 | 适用数据 |
|------|--------|---------|
| AKShare API | 东方财富、同花顺、交易所 | A股、宏观、情绪数据 |
| baostock | 证券宝 | 指数PE/PB估值数据 |
| 网络爬虫 | 新浪财经、金十数据等 | 实时行情、新闻快讯 |

**情绪指标构建方法**：

```
sentiment_raw = momentum_5d × log(volume_ratio_5d)
sentiment_zscore = (sentiment_raw - rolling_mean) / rolling_std
```

**情绪等级分类**：
| z值范围 | 等级 |
|---------|------|
| z > 2 | 极度贪婪 |
| 1 < z ≤ 2 | 贪婪 |
| -1 ≤ z ≤ 1 | 中性 |
| -2 ≤ z < -1 | 恐惧 |
| z < -2 | 极度恐惧 |

**情绪分布**：
- 中性：2,234条 (85.6%)
- 贪婪：141条 (5.4%)
- 恐惧：123条 (4.7%)
- 极度贪婪：68条 (2.6%)

## 七、使用建议

### 7.1 数据加载

```python
import pandas as pd

# 加载整合数据
df = pd.read_csv('real_data_complete.csv', parse_dates=['date'])

# 加载原始数据
hs300 = pd.read_csv('real_data/01_指数数据/hs300_daily.csv')
```

### 7.2 回归分析示例

```python
from sklearn.preprocessing import StandardScaler
import statsmodels.api as sm

# 准备变量
y = df['abs_ar']
X = df[['ivix', 'sentiment_zscore', 'cpi', 'm2_growth', 'epu',
        'volatility_20d', 'north_flow', 'margin_balance']]

# 标准化
scaler = StandardScaler()
X_scaled = pd.DataFrame(scaler.fit_transform(X), columns=X.columns)

# 回归
X_scaled = sm.add_constant(X_scaled)
model = sm.OLS(y, X_scaled).fit(cov_type='HAC')
print(model.summary())
```

### 7.3 注意事项

1. **宏观指标时滞**：月度/季度数据存在发布时滞，使用时需注意
2. **前向填充**：宏观数据已前向填充，避免未来信息泄露
3. **标准化建议**：回归前建议对自变量进行标准化
4. **情绪指标**：sentiment_zscore已标准化，可直接使用
