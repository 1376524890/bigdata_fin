### 情绪指标构建方法:
1. **动量 × 成交量** = sentiment_raw
2. **滚动标准化** = sentiment_zscore
3. **等级分类**:
   - z > 2: 极度贪婪
   - 1 < z ≤ 2: 贪婪
   - -1 ≤ z ≤ 1: 中性
   - -2 ≤ z < -1: 恐惧
   - z < -2: 极度恐惧

- 恐惧: 123条 (4.7%)
- 极度贪婪: 68条 (2.6%)

## 六、使用建议

### 1. 回归分析
```python
import pandas as pd
df = pd.read_csv('real_data_complete.csv')

# 因变量
y = df['abs_ar']

# 自变量（建议）
X = df[['ivix', 'sentiment_zscore', 'cpi', 'm2_growth', 'epu',
        'volatility_20d', 'north_flow', 'margin_balance']]

# 标准化
from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
```

### 2. 情绪指标使用
- `sentiment_zscore`: 连续变量，可直接用于回归
- `sentiment_level`: 分类变量，可转为虚拟变量
- 正值表示贪婪情绪，负值表示恐惧情绪

### 3. 注意事项
- 宏观数据使用前向填充，注意发布时滞
- 情绪指标基于市场数据计算，非直接爬取
- 建议对变量进行标准化处理
