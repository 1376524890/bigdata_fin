# MIDAS模型实验流程

本目录包含重构后的MIDAS模型两阶段实证分析代码，按实验阶段划分为5个独立模块。

## 目录结构

```
experiment_pipeline/
├── README.md                    # 本说明文件
├── main.py                      # 主运行脚本
├── 01_data_fetch.py             # 阶段1: 数据获取
├── 02_data_integration.py       # 阶段2: 数据整合
├── 03_data_analysis.py          # 阶段3: 数据分析
├── 04_midas_experiment.py       # 阶段4: MIDAS实验 (核心)
└── 05_generate_plots.py         # 阶段5: 生成图表
```

## 各阶段功能说明

### 阶段1: 数据获取 (01_data_fetch.py)

**功能**: 从各类数据源获取原始数据

**获取的数据包括**:
- **指数数据**: 沪深300、上证指数、深证成指、创业板指
- **宏观数据**: GDP、CPI、PPI、M2、社融、LPR、EPU指数
- **市场情绪**: iVIX(50ETF/300ETF)、北向资金、融资融券余额、投资者账户
- **汇率数据**: 美元兑人民币汇率
- **情绪指标**: 基于收益率、成交量、波动率计算的综合情绪指数

**输出**: `real_data/` 目录下的各类CSV文件

---

### 阶段2: 数据整合 (02_data_integration.py)

**功能**: 将各类数据整合为分析数据集

**主要操作**:
- 合并指数、情绪、宏观数据
- 计算市场收益率和异常收益率（相对于上证指数）
- 计算Amihud非流动性指标
- 将月度宏观数据前向填充到日度数据
- 数据清洗（缩尾、填充缺失值）

**输出**: `real_data_complete.csv` - 完整分析数据集

---

### 阶段3: 数据分析 (03_data_analysis.py)

**功能**: 生成描述统计和相关性分析

**分析内容**:
- 构造未来5日和60日累计收益率
- 描述性统计（均值、标准差、偏度、峰度）
- 缺失值统计
- 相关性分析：
  - 宏观变量间相关性
  - 市场状态变量间相关性
  - 与未来收益的相关性
- 训练集/测试集划分（60%/40%）

**输出**:
- `desc_stats.csv` - 描述统计
- `missing_stats.csv` - 缺失值统计
- `*_correlation.csv` - 相关性矩阵

---

### 阶段4: MIDAS实验 (04_midas_experiment.py) ⭐ 核心

**功能**: 执行MIDAS模型两阶段实证分析

#### 第一阶段: MIDAS回归预测收益率

1. **单变量MIDAS模型**
   - 对每个宏观变量（CPI、PPI、M2、EPU、USD/CNY）分别建立MIDAS回归
   - 计算样本内和样本外R²
   - 识别最优单变量

2. **VIF检验**
   - 检验多重共线性
   - 仅保留VIF < 10的变量

3. **精简多变量MIDAS模型**
   - 基于VIF筛选结果构建多变量模型
   - 计算样本外R²和RMSE

#### 第二阶段: 分组嵌套回归

1. **模型I: 情绪与风险感知**
   - 变量: iVIX
   - 方法: OLS with HAC稳健标准误

2. **模型II: 加入资金与杠杆交易**
   - 新增变量: 北向资金净流入、融资融券余额
   - 联合显著性检验（F检验）

3. **模型III: 完整模型**
   - 新增变量: Amihud非流动性、日内振幅
   - 联合显著性检验

4. **LASSO变量筛选**
   - 5折交叉验证选择最优惩罚参数λ
   - 识别保留的非零系数变量

5. **综合模型**
   - 基于LASSO结果的最终模型
   - 计算调整后R²

**输出**:
- `stage1_results.csv` - 第一阶段结果
- `stage2_results.csv` - 第二阶段结果
- `full_data_with_predictions.csv` - 含预测值的完整数据

---

### 阶段5: 生成图表 (05_generate_plots.py)

**功能**: 生成论文所需的所有图表

**生成的图表**:
1. `true_pred_5d.png` / `true_pred_60d.png` - 真实值vs预测值对比
2. `residual_stage1.png` - 第一阶段残差诊断（时间序列、散点图、直方图）
3. `residual_stage2.png` - 第二阶段残差诊断
4. `lasso_cv.png` - LASSO交叉验证结果
5. `lasso_path.png` - LASSO系数路径图
6. `marginal_r2.png` - 分组嵌套回归边际解释力变化

**输出**: `experiment_results/figures/` 目录下的PNG图表

## 使用方法

### 1. 运行所有阶段

```bash
cd /home/marktom/bigdata-fin/experiment_pipeline
python main.py
```

### 2. 运行指定阶段

```bash
# 只运行第4阶段 (MIDAS实验)
python main.py --stage 4

# 从第3阶段开始运行
python main.py --from 3
```

### 3. 查看阶段列表

```bash
python main.py --list
```

### 4. 单独运行某个阶段

```bash
python 01_data_fetch.py
python 02_data_integration.py
python 03_data_analysis.py
python 04_midas_experiment.py
python 05_generate_plots.py
```

## 关键变量说明

### 被解释变量
- `R_5d` / `R_60d`: 未来5日/60日累计收益率
- `AbsAR_5d` / `AbsAR_60d`: 异常收益绝对值（衡量预测偏离程度）

### 宏观变量 (月度)
- `cpi`: CPI同比增速
- `ppi`: PPI同比增速
- `m2_growth`: M2同比增速
- `epu`: 经济政策不确定性指数
- `usd_cny`: 美元兑人民币汇率

### 市场状态变量 (日度)
- `ivix`: 中国波指（隐含波动率指数）
- `north_flow`: 北向资金净流入
- `margin_balance`: 融资融券余额
- `amihud`: Amihud非流动性指标
- `intraday_range`: 日内振幅
- `sentiment_zscore`: 情绪标准分

## 注意事项

1. **数据依赖**: 运行阶段2-5前需要先完成阶段1的数据获取
2. **内存需求**: 阶段4实验计算量较大，建议至少4GB内存
3. **运行时间**: 完整流程约需5-10分钟（取决于网络速度和计算性能）
4. **结果复现**: 设置随机种子确保LASSO结果可复现
